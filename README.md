# Electron, Typescript and React

An Electron boilerplate with Typescript and React.

## Usage

```bash
 $ git clone https://github.com/mateusbp/electron-react-ts.git [ProjectName]
 $ cd [ProjectName]
 $ yarn install / or / npm install
```

## Run

> development

```bash
 $ yarn dev
```

or

```bash
 $ npm run dev
```
